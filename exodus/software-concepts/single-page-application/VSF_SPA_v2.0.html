INJECT VSF - DELTA UPDATE
Load a .vsf file to append its rows to the server object. This is a delta update - existing rows are preserved, new rows are added. The header in the file is validated against the current geometry but does not overwrite it. This simulates receiving state from another client, a backup, or a prior session.
